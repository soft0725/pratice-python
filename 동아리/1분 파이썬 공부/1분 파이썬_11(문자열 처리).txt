2 + '이' = 불가능 이였다

그런데 2를 형 변환을 시켜서 '2' + '이'를 하면 '2이'가 가능해짐 

문자열 처리
name = '자동차 번호'
number = '123'

자동차 번호123을 만들고 싶다면 
result = name + number을 하면 된다

print(result)
실행결과 
자동차 번호123

만약에 다르게 자동차 번호456 을 출력하고 싶다면 
result = name + '456'을 하면 된다 

또 다른 방법으로는 그냥 
name += '456'
print(name) 
을 하면 된다

+= 처럼 사용해 보자 
num = 3
num += 2를 하면 num = 5

num -= 1을 하면 2
num *= 2를 하면 6
num /= 4를 하면 2.0

문자열 길이 구하는 법 
name = '김병찬'
print(len(name))
을 하면 3이 출력된다

또 때로는 여러 줄에 해당하는 문자열이 필요한 경우도 있음 
name = '''자동차는 
너무 
멋있어요'''
