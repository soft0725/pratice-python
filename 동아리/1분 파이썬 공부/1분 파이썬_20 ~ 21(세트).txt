세트는 리스트, 튜플과 같이 여러 개의 데이터를 저장할 수 있다

세트는 값들의 순서가 보장되지 않으며, 중복이 허용되지 않는다

세트의 선언 방법
세트 = {값1,값2}

A = {'돈가스', '피자'}
B = {'치킨', '피자'}
A와 B중 공통인 것은 
intersection()을 사용하여 확인 할 수 있음(교집합)

print(A.intersection(B))
실행결과 : {'피자'}

이것을 다 합치려면 
union()을 사용(합집합)

print(A.union(B))
실행결과 : {'돈가스', '피자', '치킨'}

세트는 순서대로 출력하지 않기 때문에 출력할 때 마다 값이 바뀜
또 중복은 허용이 되지 않기 때문에 피자는 한 번만 출력 된다

A만 좋아하는 음식을 출력하면 
difference()를 사용 (차집합)
print(A.difference(B))
실행결과 : {'돈가스'}

세트는 인덱스를 통해 값 접근 불가 
값 수정 불가 
이것이 리스트와 튜플의 차이점 

하지만 세트는 값을 추가 할 수 있음
A = {'돈가스'}
A.add('민트초코')
print(A)
실행결과
{'돈가스', '민트초코'}

또 값을 제거할 수 있음
A.remove('민트초코')
print(A)
실행결과
{'돈가스'}

모든 값을 제거하려면 
A.clear()
print(A)
실행결과
set()
